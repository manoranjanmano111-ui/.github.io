GRANULE SoC Verification

GRANULE is a wearables SoC comprising multiple subsystems including processor, Bluetooth, and power management units. The design integrates these components to enable efficient audio processing and system functionality.

  • Verified OTP memory functionality
  • Validated PINMUX signal routing
  • Verified GPIO controller behavior
  • Performed regression & debugging

DEN(Dispatch Engine Verification)

DEN is an IP that acts as an interface between the processor and the DMA controller. It manages work requests by queuing and dispatching transactions within the HELLCAT/TRITON chiplet architecture. Work requests are transferred as 1024-bit data packets between processor and DMA.

  • Developed UVM testbench for DMA descriptors
  • Verified multi-cluster scheduling
  • Handled complex coverage scenarios
  • Debugged corner cases

PCIe Gen4 Protocol Verification

Worked on PCIe Gen4 Verification IP supporting 16 GT/s per lane for high-speed back-to-back communication.

  • Verified LTSSM state transitions and link training sequences
  • Handled TLP, DLLP, and Ordered Sets (TS1/TS2)
  • Debugged protocol mismatches across Transaction and Data Link layers
  • Ensured link stability and compliance through waveform analysis
